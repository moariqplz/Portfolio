## Features

- **Habit Tracking**: Users can add habits, mark them as completed or missed, and track their progress over time.
- **Progress Visualization**: The progress dashboard displays a bar chart that visualizes the number of times each habit was completed or missed.
- **Local Storage**: All data is saved in the browser's local storage, so it persists across sessions.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository** to your local machine:
    ```bash
    git clone https://github.com/yourusername/habit-tracker.git
    ```
2. **Navigate to the project directory**:
    ```bash
    cd habit-tracker
    ```
3. **Open the project** in your preferred code editor.

4. **Open the `index.html` file** in your web browser to start using the application.
